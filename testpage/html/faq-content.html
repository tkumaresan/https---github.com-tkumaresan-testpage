FAQ

Most frequently asked question's:

  • What does the application do NGP?

    Applications allows users to add companies they want to track. Once the user fills in required information, the application automatically finds the social media page for the relevant company and extracts required information. Thereafter it extracts information each month.

  • Home Page

    To see historic data users can click on ‘+’ on the left of Employees, Alexa, FB likes etc.

  • Add Company

    • User can add a company using the ‘Add Company’ Button. Users need to fill in all the necessary fields.

    • Bulk Upload: Users can bulk upload companies using excel input in the attached format.

    • Each week all new companies added to Salesforce will be added to the application (no intervention from users needed).

    • By default, it includes NGP portfolio and pipeline companies.

  • Edit Company

    As mentioned above, the application automatically finds the social media page for the relevant company. However, it is possible that the social media page is incorrect.

    The application provides a feature where users can correct it using a flag button right next to company name on the homepage. Users should then go to the Red Flagged Company list to rectify the link. On the left hand side of the company name user can click the Edit Button. On the pop up that comes up, follow the instructions to change the social media id:


    • LinkedIn: ‘Whatsapp-Inc.’ from the URL http://www.linkedin.com/company/whatsapp-inc

    • Crunchbase: ‘XX’ in from the URL: http://www.crunchbase.com/organization/XX

    • Google Play Store: ‘com.olacabs.customer’ in the Google Play Store link of the company: https://play.google.com/store/apps/details?id=com.olacabs.customer

    • FB: ‘WhatsappOfficial’ from the URL https://www.facebook.com/whatsapp.official

    • Twitter: ‘Maginteractive’ in the Twitter link of the company: https://twitter.com/maginteractive
