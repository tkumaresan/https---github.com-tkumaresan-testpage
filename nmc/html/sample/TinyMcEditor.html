AngularUI - TinyMCE Implementation

Pre-requesites:

Plugins used:

  • angular.min.1.0.8.js
  • tinymce.js
  • tinymce.min.js
  • ui-bootstrap-tpls-0.4.0.js

CSS Used:

  • bootstrap-combined.min.css

Customized js:

  • tinyMcController.js

Note:

  • This tiny.min.js has the reference of its own css and images in it, so it can be directly referenced from its original location rather pointing locally.
  • This plugin implementation is supported by older version of angular script files
  • The script files should be loaded with the same order mentioned above
