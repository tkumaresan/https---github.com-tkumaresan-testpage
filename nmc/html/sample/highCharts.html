Input details to the charts:

  • It accepts a json object as input with the following structure :
  • The title should be a string which will be display as the Title of the Chart.
  • The subtitle should be a string which will be display as the Sub - Title of the Chart.
  • The type should be a string (mostly 'bar','pie','line',etc.,). Please refer the link chart types for more info.
  • The chartSeries should be an array of json objects where the json object have the following
  • The name should be a string.
  • The data should be an array of integers
  • Ex: "chartSeries": [ { "name" : "Greg", "data" : [ 1,2,4 ] }, { "name" : "Wolf", "data" : [ 3,5,6], }, { "name" : "Steve", "data" : [ 4,6,6 ] }],
  • The exportFeature should be a boolean where it will switch on / off the exporting feature.
